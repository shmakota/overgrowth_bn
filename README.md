# Overgrowth for Bright Nights

Adds a mapgen post-processing effect using LUA that weathers terrain: smashes windows, patches of grass in the road and houses, destroyed fences and doors, and more! Intended for usage in No Hope style playthroughs with lowered item spawnrates and destroyed vehicles.
